# Release Notes - 2025-12-13

## New Features
- **Multi-Strategy Support**: Implemented a Strategy Library with 10+ strategies (RSI, MACD, Bollinger Bands, etc.).
- **Backtesting Engine**: Added `BacktestService` and `BacktestController` to simulate strategies against historical data.
- **AI Integration**: Added `LlmService` for generating daily insights and analyzing market conditions using Ollama.
- **Telegram Enhancements**: Added commands `/strategy`, `/golive`, `/backtosim`, `/talk`, `/insight`.
- **Refactored Trading Engine**: Decoupled dependencies using `TradingContext` for better maintainability and testing.
- **Dynamic Configuration**: Strategies and trading modes can be switched dynamically via Telegram.

## Improvements
- **Test Coverage**: Added comprehensive unit tests for new services and strategies.
- **Code Quality**: Refactored `TradingEngine` constructor to reduce complexity.
- **Documentation**: Updated README with new commands and architecture details.

## Fixes
- Fixed `TradingEngine` constructor having too many arguments.
- Fixed Lombok compilation issues in various services.
- Fixed NPE in `BacktestController`.
